# Instalação

## Instalação rápida (Linux / macOS)

Execute o script de instalação com `curl`:

```bash
curl -fsSL https://raw.githubusercontent.com/nitoba/pr-tools/main/apps/cli-go/install.sh | bash
```

O script baixa o binário `prt` para `~/.local/bin/prt` e configura as permissões automaticamente.

## Instalar uma versão específica

Veja as versões disponíveis em [Releases](https://github.com/nitoba/pr-tools/releases):

```bash
curl -fsSL https://raw.githubusercontent.com/nitoba/pr-tools/main/apps/cli-go/install.sh | INSTALL_VERSION=v3.4.0 bash
```

## Windows

No Windows, baixe o executável diretamente na página de [Releases](https://github.com/nitoba/pr-tools/releases). Não é necessário WSL ou Git Bash — o `prt` é um binário nativo para Windows.

## Pré-requisitos

Nenhuma dependência adicional é necessária. O `prt` é um binário compilado que inclui tudo que precisa para funcionar.

Você precisa apenas de API key de pelo menos um provider:

- [OpenRouter](https://openrouter.ai) — plano gratuito disponível
- [Groq](https://console.groq.com) — plano gratuito disponível
- [Google Gemini](https://aistudio.google.com) — plano gratuito disponível
- Ollama — local ou via Ollama Cloud

<Callout type="info">
  Ao contrário da versão anterior em Bash, o `prt` não requer `jq`, `curl` nem versão específica de shell. Funciona diretamente em qualquer sistema operacional suportado.
</Callout>

## Verificar instalação

Após a instalação, verifique se o binário está disponível:

```bash
prt --version
```

Se o comando não for encontrado, adicione `~/.local/bin` ao seu PATH:

```bash
# Adicione ao ~/.bashrc, ~/.zshrc ou equivalente
export PATH="$HOME/.local/bin:$PATH"
```

## Atualizar

Para atualizar para a versão mais recente, execute o script de instalação novamente:

```bash
curl -fsSL https://raw.githubusercontent.com/nitoba/pr-tools/main/apps/cli-go/install.sh | bash
```
